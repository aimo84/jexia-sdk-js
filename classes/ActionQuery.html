Description

Query object specialized for generic action statements. This object is generated automatically from the Dataset object, never to be instantiated directly.

Example

```typescript

  • // Attaching resources to a dataset
  • dataModule.dataset("posts")
  • .attach("comments", field => field("post.id").isEqualTo(somePostId))
  • .execute();
  • ```
